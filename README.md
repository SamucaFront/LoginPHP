# Calculadora de IMC

Este projeto tem como objetivo calcular o **Índice de Massa Corporal (IMC)** a partir do peso e altura informados pelo usuário.  
Foi desenvolvido utilizando **HTML**, **CSS** e **PHP**.

---

## Funcionamento

O usuário insere seu **peso (kg)** e **altura (m)** no formulário.  
Ao enviar os dados, o sistema calcula o IMC com a fórmula:


Em seguida, exibe o resultado e a classificação correspondente:

| Faixa de IMC | Classificação             |
|---------------|---------------------------|
| Abaixo de 18.5 | Abaixo do peso            |
| 18.5 a 24.9    | Peso ideal                |
| 25.0 a 29.9    | Levemente acima do peso   |
| 30.0 a 34.9    | Obesidade leve            |
| 35.0 a 39.9    | Obesidade moderada        |
| 40.0 ou mais   | Obesidade grave           |

---

## Tecnologias utilizadas

- **HTML5** – estrutura da página  
- **CSS3** – estilização e layout  
- **PHP** – lógica e cálculo do IMC  

---
